### 틱택토 게임 또는 노트앱 만들기

#### 틱택토 게임

##### 게임진행방식

- 👾와 👻 두 유저중 가로,세로,대각선 중 3개의 칸을 연속적으로 채우면 우승
- 9칸의 칸을 다 채웠지만 승자가 없으면 무승부

##### 느낀점

- 기록저장까지는 구현하지 못했습니다. 과제 제출 후에 추가로 구현해보겠습니다.
- 수업을 완전히 이해하지 못한 상태에서 시작해 부족한 점들이 많지만 복습에 의미를 두며 과제를 수행했습니다.
- 상태끌어올리기가 반복되면서 코드가 복잡해지며 가독성이 떨어진다고 생각했습니다.
- 수업시간에 진행했던 내용의 커밋을 보며 순서과 과정에 따라 스스로 구현해 보려하였으나 코드로직이 떠오르지 않아 복습느낌으로 코드를 따라치기만 한점이 아쉽습니다.
